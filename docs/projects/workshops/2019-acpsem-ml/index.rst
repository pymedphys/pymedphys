=====================================
ACPSEM Machine Learning Workshop 2019
=====================================

http://www.physics.usyd.edu.au/medical-physics/


In this workshop the PyMedPhys team provided a problem driven tutorial.
For each problem a set of examples were provided in Jupyter Notebook
form.

Three distinct problems were provided with the aim to cater to different
skill levels of the attendees.
